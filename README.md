# Installation

## For Desktop / PC (Linux)

```bash
> sudo apt update
> sudo apt upgrade
> sudo apt install npm
> sudo npm install -g n
> sudo n install stable
```

# Requirements

- NodeJs
- Telegram bot token
- Devices
